# portfolio
Portfolio — учебный репозиторий с решениями лабораторных и домашних работ по C/C++: реализация собственных контейнеров и структур данных, работа с памятью, матрицами, строками, исключениями, shared_ptr, параллельной сортировкой, а также отдельные задачи по BMP, Huffman coding и move semantics. Репозиторий организован по темам и показывает практику системного и объектно-ориентированного программирования на C/C++.
