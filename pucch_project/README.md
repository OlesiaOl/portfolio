📘 Описание задания

В рамках проекта реализовано моделирование передачи контрольной информации в канале PUCCH Format 2, используемом в системах связи 4G LTE и 5G NR.

Основная задача — исследовать влияние шума на надёжность передачи небольшого числа бит (2, 4, 6, 8 или 11) с использованием помехоустойчивого кодирования.

🎯 Цель работы

Реализовать и объединить в единую систему следующие компоненты:

📦 блочный кодер
📡 QPSK модулятор
🌫️ модель канала с AWGN шумом
📊 мягкий демодулятор (LLR)
🧠 мягкий декодер (метод полного перебора)


Система имитирует реальную цепочку передачи:

случайные биты → кодирование → модуляция → шум → демодуляция → декодирование



Поддерживаемые сценарии

Программа работает в трёх режимах:

coding — кодирование и модуляция
decoding — демодуляция и декодирование
channel simulation — полная симуляция с расчётом
